Admin Commands

Command Usage Description
Kick User +admin kick @User Kicks the mentioned user from the server.
Ban User +admin ban @User Bans the mentioned user from the server.
Add Role +admin addr "Role Name" Adds the named role to the list of roles available to server users.
Delete Role +admin delr "Role Name" Deletes the named role from the list of roles available to server users.
List +admin list Returns the amount of servers Tron is currently connected to.
Initialize +admin initialize I honestly can't remember what this does...
