Feature Commands

Command Usage Description
Quote +quote Returns a random quote of interest.
Ping +ping Returns the current latency to the Tron server.
Invite +invite Returns the link to invite Tron to another server.
Stats +stats "Command Name" Returns the list of commands in descending order or the use count of provided command.
RateWaifu +ratewaifu @Waifu Rate your waifu! 0 - 10
Git +git Returns a link to the Github repository for Tron.
List Roles +listr Returns a list of available roles to join for that server.
Join Role +joinr "Role Name" Join a role that is available to you.
Leave Role +leaver "Role Name" Leave a role you're currently assigned to.
Avatar +avatar @User Returns a high quality version of the tagged users avatar.
Ship +ship @User 1 @User 2 Ships two users as a lovely item. <3
Exhentai Cookies +exhentai Returns the cookies required to gain access to Exhentai.
Reddit +reddit "Subreddit" Returns a random post from the current front page of the given subreddit.
Suggestion +suggestion "This is my suggestion, heed my call!" Stores your suggestion in the database of suggestions for the developers.
MAL Search Anime +mal search anime "Anime Name" Search MyAnimeList (MAL) for the given anime.
MAL Search Manga +mal search manga "Manga Name" Search MyAnimeList (MAL) for the given manga.
MAL User Anime +mal user anime "Username" Returns the provided MAL users anime list.
MAL User Manga +mal user manga "Username" Returns the provided MAL users manga list.
MAL User +mal user "Username" Returns the basic user profile for the provided MAL user.
